# Portal CAMP Santo André - Versão Pública (Portfólio)

Este repositório contém o recorte público do portal institucional do CAMP Santo André, focado em apresentação de front-end para portfólio.

## Estrutura publicada

- `index.html`
- `pages/` (páginas HTML públicas)
- `css/` (estilos)
- `js/` (scripts)
- `assets/` (imagens, ícones e fontes)

## Páginas incluídas

- `pages/home.html`
- `pages/institucional.html`
- `pages/empresa.html`
- `pages/programas.html`
- `pages/projetos.html`
- `pages/jovem.html`
- `pages/aprovados.html`
- `pages/auxilio-transporte.html`
- `pages/bem-estar.html`
- `pages/cesta-alimentos.html`
- `pages/kits-higiene.html`
- `pages/contato.html`
- `pages/transparencia.html`

## Observações importantes

- Esta é uma versão **somente pública/visual**.
- Arquivos de backend (`.php`), área administrativa, integrações com banco, e configurações sensíveis foram removidos por segurança.
- O objetivo deste repositório é demonstrar estrutura, navegação e interface do projeto.
